# herocast

Power user tools for decentralized social. No algorithms. You own your data.

<!-- TODO: Add demo GIF/video here -->

## Why

Social media in 2025 extracts attention and sells it. herocast is built for a different future. Protocols are permissionless, clients are open source, and users control their experience.

## Features

- **Multi-account management** — Switch between Farcaster accounts seamlessly
- **Scheduled posts** — Queue content and publish on your schedule
- **Analytics** — Track engagement without invasive tracking
- **Desktop + Web** — Native apps for macOS, Windows, Linux, plus web access
- **Keyboard-first** — Power user shortcuts throughout
